# ☕ Java Basics Practice

This project contains **simple Java programs** to understand basic concepts like **pattern printing, variables, arithmetic operations, and user input using Scanner**.

---

## 📚 Topics Covered

* Java program structure
* `main()` method
* Pattern printing
* Variables and arithmetic operations
* User input using `Scanner`

---

## 💻 Example: Sum of Two Numbers

```java
import java.util.*;

public class main {
   public static void main(String[] args) {

      Scanner sc = new Scanner(System.in);

      int a = 6;
      int b = 5;

      System.out.println(a + b);
   }
}
```

### Output

```
11
```

---

## ⚙️ Run the Program

Compile:

```
javac Main.java
```

Run:

```
java Main
```

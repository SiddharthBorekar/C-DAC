# Database Technology ✅

## DML Commands
Select data using:
- `WHERE` clause
- `LIMIT` clause
- `ORDER BY` clause

- `course`

## Task

- Retrieving student details
- Retrieving faculty details
- Course and module information
- Batch and student mapping
- Student and faculty contact information
- Qualification records
- Student card and order information

```sql
